<h1>匯率轉換工具(簡易版)</h1>

<h2>可轉換幣別</h2>	

1. USD	(美金)
2. TWD	(台幣)
3. RMB	(人民幣)
4. JP	(日幣)
5. KRW	(韓元)
6. SGD	(新加坡幣)
7. EU	(歐元)
8. GBP	(英鎊)
9. AUD 	(澳幣)


<h2>操作</h2>

```
python Exchange_Rate.py
```
1. 先輸入要被轉換的幣別	(Which currency you want to be converted?)
2. 輸入金額 				(How much you want to convert?)
3. 再輸入要轉換成哪種幣別	(Which currency you want to covert to ?)
